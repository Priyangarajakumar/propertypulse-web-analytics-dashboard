# Power BI Dashboard Implementation Guide
1. Theme integration: `assets/branding/PropertyPulseTheme.json` applied via
   **Home > Switch Theme > Import theme**
2. Dataset structure: 7 key dimensions (date/page/city/device) with 5 metrics 
   (sessions/conversions/bounce_rate/etc.)
3. Key visual hierarchy:
   - Executive summary (cards + KPI strip)
   - Temporal analysis (line charts)
   - Geospatial heatmaps
